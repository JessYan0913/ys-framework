# YS Framework应用权限申请与使用情况说明

为保障 **YS Framework** 产品功能的实现及系统的安全稳定运行，我们可能需要申请或使用操作系统的相关权限。为增强您的知情权，我们将产品可能申请或使用的权限进行说明，并提供管理建议：

- 您可根据实际需要，对相关权限进行授权或管理。
- 随着产品升级，权限类型及使用目的可能发生变化，我们会及时更新权限列表，确保您及时了解权限的申请与使用情况。
- 为满足业务及产品功能需求，我们可能使用第三方 SDK，这些第三方可能会申请或使用相关操作系统权限。
- 在使用产品过程中，您可能访问第三方 H5 页面或小程序，这些第三方开发的插件或小程序也可能因业务功能所需申请或使用相关权限。

---

### 安卓操作系统应用权限列表

| **权限名称**                             | **功能说明**           | **使用场景/目的**                                          |
| ---------------------------------------- | ---------------------- | ---------------------------------------------------------- |
| CAMERA（摄像头）                         | 使用相机拍摄照片和视频 | 用于帮助您完成拍摄、上传及发送图片等功能                   |
| WRITE_EXTERNAL_STORAGE（写入外置存储器） | 写入外部存储           | 适用于 Android 10 及以下系统，用于下载、保存图片及相关文件 |
