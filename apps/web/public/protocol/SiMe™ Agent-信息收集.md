## 个人信息收集清单

| **收集场景**                       | **收集目的**                                   | **收集方式**           | **个人信息类型**                                             |
| ---------------------------------- | ---------------------------------------------- | ---------------------- | ------------------------------------------------------------ |
| 账号注册、登录、认证               | 用于注册、登录                                 | 用户主动提供           | 手机号码/邮箱地址                                            |
|                                    | 用于第三方登录                                 | 用户授权提供           | 第三方平台的账号信息                                         |
| 智能对话                           | 用于对话功能                                   | 用户主动提供           | 输入内容（包括文本、图片、文件）                             |
|                                    | 用于反馈评价                                   | 用户主动提供           | 反馈信息                                                     |
|                                    | 用于模型训练和服务优化                         | 用户主动提供           | 输入内容和对应的输出                                         |
| 开放平台服务（仅使用开放平台涉及） | 用于个人身份认证                               | 用户主动提供           | 真实身份信息（真实姓名、证件号码、电话号码及其他必要身份信息） |
|                                    | 用于企业身份认证                               | 用户主动提供           | 所在企业法定代表人、企业联系人的真实姓名                     |
|                                    | 用于推送动态通知                               | 用户主动提供           | 邮箱地址                                                     |
|                                    | 用于提供下单、支付等服务                       | 系统自动采集           | 支付订单和交易信息                                           |
| 意见反馈                           | 用于处理反馈信息                               | 用户主动提供或授权提供 | 问题描述信息、证明材料附件、手机号码、邮箱、联系人姓名、使用信息、账号信息、沟通记录 |
| 消息通知                           | 用于发送通知                                   | 用户主动提供           | 手机号码、邮箱                                               |
| 运营与安全保障                     | 用于保障软件与服务的安全运行、运营的质量及效率 | 系统自动采集           | 硬件型号、操作系统版本号、设备标识符（Android如AndroidID、OAID，iOS如IDFV；不同的标识符在有效期、是否可由用户重置以及获取方式方面会有所不同）、IP 地址、软件版本号、网络接入方式、类型、状态、网络质量数据、操作、使用、服务日志、设备传感器数据（如加速度传感器、陀螺仪传感器） |
|                                    | 用于预防恶意程序、保障运营质量及效率           | 系统自动收集           | 正在运行的进程信息、页面的总体运行、使用情况与频率、页面崩溃情况、总体使用情况、性能数据、账户信息、设备信息以及服务日志信息 |
